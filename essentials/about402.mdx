## What it is 



> Built around the HTTP 402 status code, x402 enables users to pay for resources via API without registration, emails, OAuth, or complex signatures.


### X402 Features


```md
No fees
Instant settlement
Blockchain Agnostic
Frictionless
```


### Velocity powered X402 

```x402 requires client and server setup for devs  to enable the protocol```
> Velocity makes it a no brainer by auto x402 monetization of your endpoints registered with velocity
